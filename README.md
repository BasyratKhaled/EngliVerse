# ✨ EngliVerse - Interactive English Learning Platform

EngliVerse is a fully responsive, gamified front-end web application designed to help beginners master the English language through interactive play. It features animated lessons, dynamic flashcards, and a responsive word scramble puzzle, all wrapped in a modern glassmorphism UI.

## 🚀 Features

* **Gamified Learning:** * **Interactive Flashcards:** 3D flip animations to test vocabulary and definitions.
  * **Word Scramble Puzzle:** A dynamic, randomized anagram game with live score tracking.
* **In-Depth Lesson Modules:** Expandable accordion-style lesson pages detailing grammar, vocabulary, greetings, and phonics without cluttering the screen.
* **Modern UI/UX:**  Floating animated background bubbles using pure CSS/JS.
  * Glassmorphism design elements (translucent, blurred backgrounds).
  * Smooth scroll-reveal animations for all sections.
* **Zero Dependencies:** Built entirely with vanilla web technologies. No external libraries or frameworks required.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, flexbox/grid, CSS animations (`@keyframes`), and transitions.
* **Vanilla JavaScript (ES6+):** DOM manipulation, game logic, randomization algorithms, and event handling.
